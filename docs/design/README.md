# Design

Versemark's visual identity is a calm daily ritual: a verse on a card, a canon timeline, a marker laid down. Quiet, warm, literary, fast. No image assets — everything is CSS and Canvas.

## Ground rules

- No image assets. Everything is CSS and Canvas-drawn. No AI asset pipeline, no generated images.
- The accent (terracotta) is used for the player's action only, never for decoration.
- Warmth lives in serif typography and the accent, not in a tinted background.
- The background stays near-white. No cream, sand, or parchment defaults.
